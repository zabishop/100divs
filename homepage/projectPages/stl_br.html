STL Brew Review

This is project is my first venture into mobile development utilizing the PhoneGap/Cordova framework. Currently, it is still in development.

The inner workings of this app are pretty straight forward. When first loaded, the app makes an XMLHttpRequest to stlbrewreview.com/breweries.json, parses the JSON, and assigns the details to an array. This array is then passed to a mustache template.

When a user selects a brewery, the app queries the array for the associated brewery object and passes the retrieved data to another mustache template.

The details screen for each brewery is pretty straight forward; mostly, it's a list of links that directs the user toward more information about that particular brewery.

Bet you can't guess what the "Beer" link does when you click it? That's right! It queries for "beer", parses, and renders another mustache template. You = smart.

I don't think I need to tell you what happens when you click on a beer.

That's pretty much it at the moment. A plethora of other features will (eventually) get added.
